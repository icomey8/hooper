# hooper
> Created by Ian Comey and Jaime Quiogue

## About

Designing a new, unique language for CMSI 3802

## Features

- enforced encapsulation
- static typing
- inheritance
- method overloading

## Keyword Guide

| hooper   | Javascript    |
| -------- | ------------- |
